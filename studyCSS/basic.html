css basic

Cascading Style Sheets의 약자.
HTML 요소가 화면에 표시되는 방식을 설명.

css 문법

선택자 {속성:속성값;}

css 선택자(Selector)의 종류 대표 3가지

  • tag 선택자 - tag 그대로를 선택자로 활용.
    강제적용, 자동적용.
  • id 선택자 - tag에 id를 추가하고 #id값으로 선택자로 활용.
    id값이 중복적용 불가하므로 해당 스타일도 한번만 적용가능. 주로 구조적인 용도에 쓰임.
  • class 선택자 - tag에 class를 추가하고 .class값으로 선택자로 활용.
    class값이 중복적용이 가능하므로 해당 스타일도 여러번 적용 가능. 주로 디자인적인 용도에 쓰임.
    class끼리 조합도 가능.

css 적용방식 3가지

  • External 방식 - 외부파일 방식
    확장자가 *.css로 끝나는 스타일 파일 안에 스타일 소스를 정의
    html 문서의 head 안에 <link href="*.css" rel="stylesheet">로 외부 스타일시트를 연결하여 적용.
    여러 웹페이지를 같은 디자인(통일성)으로 구성할 수 있고 수정도 가능(유지보수 유리)
  • Internal방식 - 내장 방식
    html문서의 head안에 스타일소스를 적용.
    해당 웹페이지에만 적용 됨.
  • Inline방식 - 인라인 방식
    tag뒤에 바로 스타일소스를 적용.
    소스가 길어지고 통일성 있는 디자인이 불가하며 유지보수에 불리.
    우선순위는 가장 높음.
    본문 내의 태그만 수정가능한 환경에서 쓰임.
