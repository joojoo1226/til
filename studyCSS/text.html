text- 로 시작하는 글자 속성들

  • text-align: ; 영역내의 글자 수평정렬
    • left 왼쪽에 정렬(기본)
    • center 중앙정렬
    • right 오른쪽에 정렬
    • justify 양끝정렬, 양끝맞춤
  • text-decoration: ; 글자의 줄들
  • text-indent: ; 들여쓰기
  • text-transform: ; 대소문자 지정
  • text-shadow: ; 글자의 그림자 지정
  • text-overflow: ; 영역밖의 글자 처리

text-align: ; 영역내의 글자 수평정렬

text-align:left ; 글자 수평정렬

text-align:center ; 글자 수평정렬

text-align:right ; 글자 수평정렬

text-align:justify ; 글자 수평정렬

text-decoration: ; 글자의 줄들

text-decoration: overline; 글자의 윗줄

text-decoration: line-through; 글자의 가로지르는 줄(취소선)

text-decoration: underline; 글자의 밑줄

text-decoration: none; 글자의 줄 제거
